# Tech Trends 2024

## Overview
This project showcases the most exciting tech trends of 2024 using a simple, responsive website built with HTML, CSS, and JavaScript. The site features summaries, useful links, and embedded videos highlighting cutting-edge innovations like AI, autonomous robotics, flying cars, and more.

## Features
- **Summary of 2024 Tech Trends:** Highlights key advancements in technology.
- **Useful Links:** Provides access to authoritative tech resources and news.
- **Embedded Videos:** Includes featured videos from CES 2024 and design trends.
- **Responsive Design:** Ensures a seamless experience across devices.
- **Header Animation:** Adds a dynamic animation to the header using JavaScript.

## How to Use
1. Clone the repository:
    ```bash
    git clone https://github.com/cpuncekar/tech-trends-2024.git
    ```
2. Navigate to the project directory:
    ```bash
    cd tech-trends-2024
    ```
3. Open the `index.html` file in your web browser to view the site.
